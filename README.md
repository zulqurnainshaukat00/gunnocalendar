
# Gunnocalendar v12.2 (Hybrid Weather & Agri Edition)

- English + Islamic (tabular) + Punjabi (Desi) dates (Punjabi **day-of-month included**)
- Farming guide (English + Urdu) by Punjabi month (Sahiwal zone)
- Weather: AccuWeather (with your API key) → fallback to Open‑Meteo (no key)
- Spray-wind advice in English + Urdu
- Reminders (local), export/import JSON
- PWA: install to home screen, offline cache, icons
- Themes: Light / Dark / System

## Put your AccuWeather key
Open the app → **Settings** → paste your key → Save.  
Default coordinates are your village (58/5‑L Gunnō Bhattian).

## GitHub Pages
Upload this folder to your repo (e.g., `gunnocalendar`) → Settings → Pages → Source: **Deploy from a branch**, Branch: `main`, Folder: `/ (root)`.
